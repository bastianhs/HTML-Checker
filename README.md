# HTML-Checker
Simplified HTML checker using Pushdown Automata

How to run: <br />
`git init` <br />
`git clone https://github.com/bastianhs/HTML-Checker.git` <br />
`cd HTML-Checker` <br />
`cd src` <br />
`python3 main.py` <br />
